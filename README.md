Introduction
============

This is a tiny C++ parser created by using `Flex` and `Bison` for parsing the
field specs like:

    "U:3d:keep_vert_coord"
    "calc_pt(P, T):3d"

in the father project `geodiag-cmor`. I store it here for demonstration the
usage of `Flex` and `Bison`.
